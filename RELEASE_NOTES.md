# LT Messages v0.8.5 - First Release

**Send preset squad messages with one click!**

This Blish HUD module helps Guild Wars 2 Commanders and Lieutenant Tags quickly send preset messages to their squad.

---

## What's Included

✅ **17 Default Messages** - Ready-to-use commander messages for common situations
✅ **One-Click Sending** - Click the [LT] icon, pick a message, done
✅ **In-Game Editor** - Right-click to add, edit, or delete messages
✅ **Auto-send or Clipboard** - Your choice: auto-type messages or copy to clipboard
✅ **LT Mode Toggle** - Safety feature prevents accidental sends
✅ **Optional Keybinds** - Set up hotkeys if you want (none required)
✅ **Works with Squad & Subgroup** - `/squad` or `/subgroup` chat support

---

## Installation

1. Download `bh.lt.messages_0.8.5.bhm`
2. Place in: `Documents\Guild Wars 2\addons\blishhud\modules\`
3. Restart Blish HUD
4. Enable the module

---

## Quick Start

1. Look for the **[LT] icon** in your Blish HUD corner menu
2. **Left-click** to see your message list
3. **Click any message** to send it to squad chat

**That's it!**

---

## Key Features

### Easy Message Sending
- Left-click the [LT] icon to open the message popup
- Click any message to send it
- Close with X button, ESC, or click outside

### Message Editor
- Right-click the [LT] icon to open the editor
- Add, edit, or delete messages
- Changes save automatically

### Optional Keybinds
Set up hotkeys in settings (completely optional):
- Popup keybind: Open message list
- Toggle LT Mode: Quick enable/disable
- Open Editor: Quick editor access

### Safety First
- **LT Mode** prevents accidental sends when not commanding
- Notifications warn you if LT Mode is disabled
- Character limits match GW2 chat restrictions
