# LT Messages v0.8.6

Send preset squad messages with one click! Complete commander toolkit with 30 essential messages.

## What's New in v0.8.6

**30 Comprehensive Messages** - Expanded from 17 to 30 messages covering all commanding scenarios:
- Pre-Movement & Positioning (5): Stack, Wait, Buffs, Stealth, Blast
- Movement Commands (7): Moving, Stop, Port, Portal, Unlock-WP, Take-WP, Link-WP
- Combat Actions (8): Focus, Kill-Adds, Spread, Dodge, Rez, Mount-CC, Need-CC, Safe
- Objectives (5): HP-Combat, HP-Commune, F-Vista, POI-Tag, POI-Marker
- Squad Management (5): Guard, Loot, Help, No-Drop, Break

**Restore Defaults Button** - Easily reset to default messages anytime from the message editor

**Improved Waypoint Messages** - Three distinct messages for different scenarios:
- Unlock-WP (map completion) | Take-WP (teleport) | Link-WP (ping in chat)

**Better UI** - All editor buttons on top row, expanded message list, no overlapping

**Authentic GW2 Language** - All messages use standard commander terminology

## Features

- Click-to-send messages to squad/subgroup chat
- Auto-send or clipboard mode
- In-game message editor (right-click [LT] icon)
- Customizable keybinds
- LT Mode toggle for safety
- Works with /squad and /subgroup commands

## Upgrading from v0.8.5

Fully backwards compatible! Your custom messages are preserved. Use the new "Restore Defaults" button to get the 30 new messages, or keep your existing setup.

## Quick Start

1. Left-click [LT] icon - See messages
2. Click any message to send to squad
3. Right-click [LT] icon - Open editor
4. Click "Restore" to reset to defaults
